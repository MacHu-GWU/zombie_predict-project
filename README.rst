Install
------------------------------------------------------------------------------

``zombie_predict`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install zombie_predict

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade zombie_predict
